5. Longest Palindromic Substring

Given a string s, find the longest palindromic substring in s. You may assume that the maximum length of s is 1000.

  • 思路：L[i][j] = true，表示字符串第i个位置到第j个位置为回文子串。初始化所有L[i][i]为true。且有L[i][j] = true if (s[i] == s[j] && (j - i) == 1), L[i][j] = L[i + 1][j - 1] if (s[i] == s[j])

1. Two Sum

Given an array of integers, return indices of the two numbers such that they add up to a specific target.
You may assume that each input would have exactly one solution, and you may not use the same element twice.

  • 思路：可使用一个hashmap，固定一个数，看想要的数是否在hashmap里

15. 3Sum

Given an array S of n integers, are there elements a, b, c in S such that a + b + c = 0? Find all unique triplets in the array which gives the sum of zero.

思路：首先对列表升序排序。固定第一个数，然后设置两个移动指针，一个指针从第一个数后面开始，另一个指针从尾部开始。 需要注意的是，中间要跳过重复的数。

18. 4Sum

Given an array S of n integers, are there elements a, b, c, and d in S such that a + b + c + d = target? Find all unique quadruplets in the array which gives the sum of target.

  • 思路：和3Sum一样，只是需要先固定前两个数
