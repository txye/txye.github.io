最长回文子串

最优子结构性质：假设输入的 n 个字符为 X[0..n-1]，L(0, n-1) 为 X[0..n-1] 的最长回文子串的长度；如果第一个和最后一个字符相同，那么有 L(0, n-1) = L(1, n-2) + 2 否则L(0, n-1) = MAX (L(1, n-1), L(0, n-2))。

最长公共子序列

不要求连续
假设两字符串分别为 X[0..m-1] 和 Y[0..n-1]，长度分别为 m 和 n，假设 L(X[0..m-1], Y[0..n-1]) 是字符串 X 和 Y 的 LCS 的长度，则下面递归的定义 L(X[0..m-1], Y[0..n-1])。
如果两字符串的最后一个字符相等，即 X[m-1] == Y[n-1]，则
L(X[0..m-1], Y[0..n-1]) = 1 + L(X[0..m-2], Y[0..n-2])
否则，两字符串的最后一个字符不相等，即 X[m-1] != Y[n-1]，则
L(X[0..m-1], Y[0..n-1]) = MAX ( L(X[0..m-2], Y[0..n-1]), L(X[0..m-1], Y[0..n-2])

字符串的编辑距离

字符串的编辑距离（Edit Distance），是指给定两个字符串 str1 和 str2。只能对 str1 进行三种操作
（1）插入一个字符；
（2）删除一个字符；
（3）替换一个字符；
求通过对 str1 进行操作，使得它变成 str2 的最少操作次数。
简单的办法是从左到右或从右到左遍历处理两字符串，假设 str1 的长度为 m，str2 的长度为 n。假设我们从右向左遍历。
算法描述如下
1）如果 str1 和 str2 的最后一个字符相同，则无需什么操作，递归的处理剩余的 m-1、n-1 个字符；
2）否则（即最后一个字符不相同），则考虑对 str1 进行所有的三种操作，然后递归计算每种操作最终导致的总操作次数，取最小者即可：
（a）插入一个字符，然后递归计算剩下的 m、n-1 个字符；
（b）删除一个字符，然后递归计算剩下的 m-1、n 个字符；
（c）替换一个字符，然后递归的计算剩下的 m-1、n-1 个字符。
