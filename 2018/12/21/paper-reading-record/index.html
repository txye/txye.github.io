    • Switch-based Active Deep Dyna-Q：Efficient Adaptive Planning for Task-Completion Dialog Policy Leaning

    The main contribution of this paper is dynamically determining whether to use the simulated dialogues for training. The simulated experiences are generated by a world model (can be seen as a fake user). There is a “switcher” (a model needs to be trained) to evaluate the quality of the simulated data. Only if it exceeds a threshold, it can be added to the training set. The technical points in this paper include:

    Dynamic Q-leaning

    Active learning

    Model-based

    World model
